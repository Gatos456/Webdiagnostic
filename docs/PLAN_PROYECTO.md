# WEB DIAGNOSTIC

## Objetivo

Crear un programa de diagnóstico para consolas.

Versión inicial:
- Nintendo Wii

En el futuro:
- PlayStation
- Xbox
- Nintendo Switch
- Otras consolas

## Funciones de la versión 0.1

- Abrir un archivo de diagnóstico.
- Leer la información.
- Mostrar el problema.
- Explicar la causa.
- Explicar cómo repararlo.

## Objetivo final

Crear una herramienta profesional para ayudar en el diagnóstico y reparación de consolas.
